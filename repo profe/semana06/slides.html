The Correlation Coefficient (r)

A number between -1 and +1 that measures relationship strength

-1.0
Perfect Negative -0.7
Strong Negative -0.3
Weak Negative 0
None +0.3
Weak Positive +0.7
Strong Positive +1.0
Perfect Positive

Interpreting Strength

|r| ValueInterpretation
0.7 - 1.0Strong relationship
0.3 - 0.7Moderate relationship
0.0 - 0.3Weak relationship

Dance Partner Interpretation

|r| > 0.7 = Holding hands tight, perfectly synchronized
0.3 < |r| < 0.7 = Casual partners, somewhat in sync
|r| < 0.3 = Barely dancing together

CRITICAL: Correlation Does NOT Mean Causation

THE MOST IMPORTANT LESSON IN DATA ANALYTICS

Just because two things move together does NOT mean one CAUSES the other!

Scatter Plots: Visualizing Relationships

import matplotlib.pyplot as plt import seaborn as sns # Basic scatter plot plt.figure(figsize=(10, 6)) plt.scatter(df['SUSCRIPTORES_ACUEDUCTO'], df['CONSUMO_ACUEDUCTO']) plt.xlabel('Subscribers') plt.ylabel('Water Consumption (m3)') plt.title('Subscribers vs Consumption') plt.show() # With regression line (seaborn) sns.regplot(x='SUSCRIPTORES_ACUEDUCTO', y='CONSUMO_ACUEDUCTO', data=df)

What to Look For

  • Direction: Up-right = positive, down-right = negative
  • Strength: Tight cluster = strong, scattered = weak
  • Shape: Linear? Curved? Clusters?
  • Outliers: Points far from the pattern
Pro Tip: Add a regression line to see the trend more clearly

Scatter plots are the FOUNDATION of bivariate analysis for numeric variables

Calculating Correlation in Python

Single Correlation

# Correlation between two columns r = df['SUSCRIPTORES_ACUEDUCTO'].corr(df['CONSUMO_ACUEDUCTO']) print(f"Correlation: {r:.3f}") # Output: Correlation: 0.847 # Interpretation if abs(r) > 0.7: print("Strong relationship") elif abs(r) > 0.3: print("Moderate relationship") else: print("Weak relationship")

Correlation Matrix (All Variables)

# Correlation between ALL numeric columns corr_matrix = df.corr() # Show the matrix print(corr_matrix) # Find strongest correlations # (excluding diagonal = 1.0) mask = corr_matrix != 1.0 strongest = corr_matrix[mask].abs().max() print(f"Strongest: {strongest:.3f}")

Warning: Correlation Assumptions

  • Linear relationships only: Correlation misses curved patterns
  • Sensitive to outliers: One extreme point can change r dramatically
  • Numeric data: Use different methods for categorical data

Correlation Heatmaps

See ALL correlations at once with a color-coded matrix

import seaborn as sns import matplotlib.pyplot as plt # Create correlation matrix corr_matrix = df.corr() # Create heatmap plt.figure(figsize=(10, 8)) sns.heatmap(corr_matrix, annot=True, # Show numbers cmap='RdYlGn', # Color scheme center=0, # 0 = neutral color vmin=-1, vmax=1, # Scale limits fmt='.2f') # 2 decimal places plt.title('Correlation Heatmap') plt.tight_layout() plt.show()

Reading the Heatmap

  • Dark Green = Strong positive (+1)
  • Yellow = No correlation (0)
  • Dark Red = Strong negative (-1)

Diagonal is always 1.0
(every variable correlates perfectly with itself)

Best Practice: Use annot=True to show the actual correlation values. Colors alone can be misleading!

Categorical vs Numeric Analysis

When one variable is a category (municipality, region, type)

Box Plot by Category

# Compare consumption by municipality plt.figure(figsize=(12, 6)) sns.boxplot(x='MUNICIPIO', y='CONSUMO_ACUEDUCTO', data=df) plt.xticks(rotation=45) plt.title('Consumption by Municipality') plt.show() # Compare means df.groupby('MUNICIPIO')['CONSUMO_ACUEDUCTO'].mean()

Bar Chart with Means

# Average consumption by stratum means = df.groupby('ESTRATO')['CONSUMO_ACUEDUCTO'].mean() plt.figure(figsize=(10, 6)) means.sort_values().plot(kind='barh') plt.xlabel('Average Consumption') plt.title('Consumption by Stratum') plt.show()

When to Use Box Plot

  • Want to see distribution within each group
  • Need to identify outliers by group
  • Compare medians and spread

When to Use Bar Chart

  • Just need to compare averages
  • Simpler message for stakeholders
  • Many categories to compare

Multivariate: Adding a Third Variable

Does the relationship CHANGE across different groups?

Color by Category

# Scatter with color = third variable plt.figure(figsize=(10, 6)) sns.scatterplot(x='SUSCRIPTORES_ACUEDUCTO', y='CONSUMO_ACUEDUCTO', hue='ESTRATO', # Color by stratum data=df) plt.title('Consumption vs Subscribers by Stratum') plt.show()

Correlation by Group

# Calculate correlation for each group for estrato in df['ESTRATO'].unique(): subset = df[df['ESTRATO'] == estrato] r = subset['SUSCRIPTORES_ACUEDUCTO'].corr( subset['CONSUMO_ACUEDUCTO'] ) print(f"{estrato}: r = {r:.3f}") # Output might show different patterns! # Estrato 1: r = 0.92 # Estrato 3: r = 0.45 # Estrato 5: r = 0.78

Simpson's Paradox: When Overall Hides the Truth

Overall correlation: r = 0.60 (moderate positive)
But when you split by stratum:
- Estrato 1: r = 0.90 (strong)
- Estrato 5: r = 0.20 (weak)

The third variable reveals hidden patterns!

Pair Plots: See Everything at Once

import seaborn as sns # Pair plot: all combinations sns.pairplot(df[['CONSUMO_ACUEDUCTO', 'SUSCRIPTORES_ACUEDUCTO', 'PROMEDIO_ACUEDUCTO', 'ESTRATO']], hue='ESTRATO') # Color by group plt.show() # This creates a matrix of: # - Scatter plots (off-diagonal) # - Histograms (diagonal) # - All colored by region

What You Get

  • Diagonal: Distribution of each variable
  • Upper triangle: Scatter plots
  • Lower triangle: Same scatter plots (mirrored)
  • Colors: Show groups within each plot
Warning: Can be slow with many variables! Select only the most important ones.

Pair plots are the FASTEST way to explore multiple relationships at once

Visualize plt.figure(figsize=(8, 6)) if df[var1].dtype in ['int64', 'float64']: sns.scatterplot(x=var1, y=var2, data=df) else: sns.boxplot(x=var1, y=var2, data=df) # 3.
