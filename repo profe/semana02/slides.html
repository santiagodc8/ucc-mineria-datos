Cargando Datos: read_csv

import pandas as pd # Forma basica df = pd.read_csv('estados_financieros.csv') # Con opciones comunes df = pd.read_csv( 'estados_financieros.csv', encoding='utf-8', # Para tildes y enes sep=';', # Separador (a veces es ;) decimal=',' # Decimales con coma )

Problemas comunes

  • Encoding incorrecto (caracteres raros)
  • Separador incorrecto (todo en una columna)
  • Decimales con coma (numeros como texto)

Tip: Abrir en VSCode primero

Antes de cargar, abre el CSV en VSCode para ver:

  • Cual es el separador?
  • Tiene encabezados?
  • Como son los numeros?

Seleccionando Datos

Una columna (Series)

# Notacion de corchetes df['nombre_columna'] # Notacion de punto (si no hay espacios) df.nombre_columna

Multiples columnas (DataFrame)

# Lista de columnas df[['col1', 'col2', 'col3']]

Filas por posicion

# Fila 0 (primera) df.iloc[0] # Filas 0 a 4 df.iloc[0:5] # Ultima fila df.iloc[-1]

Filas y columnas

# Filas 0-4, columnas 1-3 df.iloc[0:5, 1:4] # Por nombre de columna df.loc[0:5, ['col1', 'col2']]

iloc = por posicion (i de integer) | loc = por etiqueta (label)

Filtrando Datos

*

Filtrar es como buscar en Amazon: Quiero productos con precio menor a $50,000 Y categoria "Electronica"

Filtro simple

# Valores mayores a 1 millon df[df['valor'] > 1000000] # Categoria especifica df[df['categoria'] == 'Activo']

Multiples condiciones

# AND: ambas condiciones df[(df['valor'] > 1000000) & (df['categoria'] == 'Activo')] # OR: cualquier condicion df[(df['anio'] == 2023) | (df['anio'] == 2024)]

Error comun

# MAL: usar and/or de Python df[df['a'] > 5 and df['b'] < 10]

Forma correcta

# BIEN: usar & y | con parentesis df[(df['a'] > 5) & (df['b'] < 10)]
