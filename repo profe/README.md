## Estructura del Repositorio

Cada semana tiene su carpeta con los materiales de clase:

```
semanaXX/
├── slides.html              # Presentación de la clase
├── exercise.ipynb           # Ejercicio para hacer en clase
└── workshop/
    ├── README.md            # Instrucciones del taller
    └── workshop_starter.ipynb  # Notebook inicial del taller
```

## Cómo Usar

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/TU-USUARIO/analitica-datos-estudiantes.git
   ```

2. **Abre los notebooks en Jupyter:**
   ```bash
   cd analitica-datos-estudiantes
   jupyter notebook
   ```

3. **O visita el sitio web:** [https://TU-SITIO.netlify.app](https://TU-SITIO.netlify.app)
